Who is this guide for?

This guide is designed for developers, solution architects, and business professionals who want to:

  • Integrate Viettel APIs into their applications.
  • Set up and deploy Viettel's API-based products and services.
  • Manage existing connectivity and access support for Viettel's APIs.

What you will learn

This document will guide you:

  • How to sign up for a free developer account to access test and live environments.
  • How to explore APIs, test them, build a sandbox app, and use Postman collections.
  • How to obtain access to live services and secure production credentials.
  • How to get development support and connect with the Dev team.

Explore our site

The Vodafone Developer Marketplace provides a range of APIs, tools, and resources. Key highlights include:

  • Solutions: This section connects you with Vodafone’s API portfolios, showcasing their benefits and real-world use cases. Discover how these APIs can solve business challenges and bring your ideas to life.
  • APIs: This section lists available API products, categorized by their solution portfolio. Browse the API catalogue to find the best fit for your needs, or use the Postman collection from the Toolkit submenu to test them.

For each API, you’ll find:

  • A detailed overview, including its purpose and potential applications.
  • OpenAPI (Swagger) documentation for seamless integration.
  • Access the Sandbox for testing
  • Support: Get help quickly through the Support page, which includes a Contact Us section for direct assistance via the Developer Interest Form. The FAQs and Troubleshooting section offers filterable solutions to common issues, helping you resolve problems efficiently.

Account Registration

You can explore Vodafone Developer Portal APIs without signing up. However, registering for a free developer account is recommended, as it grants access to the sandbox environment and allows your organization to request production access through the portal or by contacting Vodafone.

  1. Click Register. This is located at the header of the website.
  2. Fill out the Create account fields:
    • Last name and First name
    • Phone number
    • New password and Confirm password fields
  3. Review and accept the Terms and Conditions by marking the checkbox, then click Continue.
  4. On the page that appears, click Got it. You will be logged in automatically.

Using Sandbox environment

After creating account and logging in, you are able to use the Sandbox environment. To start testing Vodafone APIs in the sandbox environment, you must create a sandbox application on the Vodafone Developer Marketplace. These applications enable you to test API features and obtain the necessary credentials, such as API keys, to use Vodafone’s Postman collections. These credentials are crucial for safely exploring and integrating APIs before moving to production. Once you’re familiar with the APIs, you can proceed with building your application for live deployment.

  1. Enter Sandbox environment
  2. Create application by clicking button CREATE AN APP
  3. Fill relevent info for application and proceed:
    • Application Name
    • Description
    • Domain used by the application
  4. In the Subscription tab, search for the API you want and subscribe to the suitable API plan.
  5. Confirm all the detail of the app you have created and click CREATE AN APP. Then your application is ready to use API.
  6. Now you can test the API function with tool like Postman. Open Postman, fill in those information shown in image and send the request. (All required information will be provided by us)
  7. From your side, you will see a Login form popup. Fill in the information to get the authorization code.
  8. Use this authorization code as exchange for Access Token. You will need to include this Access Token in later API calling step. To know more about how to use API and which methods each API support, please refer to the Documentation tab of each API.
